# Social Media Post Planner

A modern React.js application for planning and scheduling social media posts with hashtag generation capabilities.

## Features

- **Post Scheduling**: Schedule posts with specific date and time
- **Automatic Posting**: Automated posting workflow (coming soon)
- **Hashtag Planner**: Generate relevant hashtags based on keywords
- **Dark Theme**: Modern, professional dark interface
- **Responsive Design**: Works seamlessly on all devices

## Tech Stack

- React.js 18
- React Router DOM 6
- Tailwind CSS 3
- Vite

## Installation

1. Extract the ZIP file
2. Navigate to the project directory
3. Install dependencies:
   \`\`\`bash
   npm install
   \`\`\`

## Running the Application

Start the development server:
\`\`\`bash
npm run dev
\`\`\`

The application will open at `http://localhost:5173`

## Building for Production

Create a production build:
\`\`\`bash
npm run build
\`\`\`

Preview the production build:
\`\`\`bash
npm run preview
\`\`\`

## Usage

### Creating a Post
1. Navigate to "Create Post"
2. Enter your post content
3. Select date and time
4. Click "Schedule Post"

### Generating Hashtags
1. Navigate to "Hashtag Planner"
2. Enter a keyword
3. Click "Generate Hashtags"
4. Copy the generated hashtags
